# نظام إدارة الشحنات - الجهات الأربع (النسخة المحدثة)

## نظرة عامة

هذه النسخة المحدثة من نظام إدارة الشحنات للجهات الأربع تتضمن ميزات جديدة لقسم الأرشيف.

## التحديثات الجديدة (2025-11-01)

### 1. خاصية البحث في الأرشيف
- بحث متقدم في الشحنات المؤرشفة
- معايير بحث متعددة: رقم الملف، العميل، المسؤول، نطاق التاريخ
- واجهة سهلة الاستخدام مع زر مسح الفلاتر

### 2. خاصية طباعة تقرير مفصل
- تقرير شامل لجميع الشحنات المؤرشفة
- تصميم احترافي مع شعار الشركة
- يتضمن جدول ملخص وتفاصيل كاملة لكل شحنة
- جاهز للطباعة والحفظ كـ PDF

## الملفات المهمة

- `index.html` - الملف الرئيسي للنظام (محدث)
- `archive_functions.js` - الدوال الجديدة لقسم الأرشيف
- `user_guide.md` - دليل استخدام الميزات الجديدة
- `test_results.md` - نتائج اختبار الميزات الجديدة
- `README.md` - هذا الملف

## كيفية الاستخدام

1. افتح ملف `index.html` في المتصفح
2. سجل الدخول باستخدام:
   - اسم المستخدم: `admin`
   - كلمة المرور: `admin123`
3. انتقل إلى قسم "الأرشيف"
4. استخدم خاصية البحث أو طباعة التقرير

## المتطلبات التقنية

- متصفح حديث يدعم:
  - HTML5
  - CSS3
  - JavaScript ES6+
  - IndexedDB
- لا يتطلب اتصال بالإنترنت بعد التحميل الأولي

## الميزات الأساسية

- إدارة الشحنات (إضافة، تعديل، حذف، أرشفة)
- البحث والفلترة المتقدمة
- تفاصيل الشحنة الكاملة
- التقارير المتنوعة
- إدارة المراحل والمسؤولين والمنافذ
- رفع المستندات
- نسخ احتياطية للبيانات

## التقنيات المستخدمة

- HTML5
- CSS3 (مع Flexbox و Grid)
- JavaScript (Vanilla JS)
- IndexedDB للتخزين المحلي

## الدعم

للحصول على المساعدة أو الإبلاغ عن مشكلة، يرجى الرجوع إلى دليل الاستخدام أو التواصل مع فريق الدعم الفني.
